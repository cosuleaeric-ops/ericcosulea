# Elite Deep Work Blocker

Extensie Chrome pentru a bloca Facebook, Instagram, X/Twitter, Reddit si TikTok cat timp timerul `work` ruleaza in `/deep-work/`.

## Instalare

1. Deschide `chrome://extensions`
2. Activeaza `Developer mode`
3. Apasa `Load unpacked`
4. Selecteaza folderul `deep-work-extension`

## Cum functioneaza

- cand pornesti timerul pe `https://ericcosulea.ro/deep-work/`, extensia activeaza reguli de blocare
- cand opresti timerul sau intri pe `rest`, regulile se dezactiveaza

## Site-uri blocate

- Facebook
- Instagram
- X
- Twitter
- Reddit
- TikTok
